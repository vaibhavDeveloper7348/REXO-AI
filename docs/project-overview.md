# REXO AI - Project Overview

## Introduction

REXO AI is an Android-based intelligent assistant application developed using Java in Android Studio and powered by Python backend APIs.

The application combines multiple AI-driven modules into a single platform, allowing users to interact through text-based interfaces for obtaining information, performing utility operations, and analyzing security threats.

The project was developed as a practical implementation of Android application development, API integration, artificial intelligence concepts, and cybersecurity techniques.

---

## Problem Statement

Users often require quick access to information, utility services, and security analysis tools within a single platform.

Traditional applications typically provide only one service at a time.

REXO AI addresses this limitation by integrating multiple intelligent modules into one Android application.

---

## Objectives

1. Develop an Android-based AI assistant capable of handling multiple intelligent services.

2. Integrate Android frontend components with Python backend APIs using REST architecture.

3. Provide GNDEC-related information through a dedicated FAQ assistant.

4. Perform utility-based operations such as calculations, web searches, and entertainment features.

5. Detect potential SQL injection attacks through file and query analysis.

---

## Project Modules

### 1. Jack AI

Jack AI is a GNDEC-focused intelligent assistant.

Features:

* Answers college-related questions
* Provides hostel information
* Provides admission information
* Provides department details
* Provides general college guidance

The module uses a structured FAQ knowledge base stored on the backend.

---

### 2. Jarvis AI

Jarvis AI acts as a personal utility assistant.

Features:

* Mathematical calculations
* Opening YouTube searches
* Joke generation
* General utility commands
* Smart command processing

Jarvis improves user productivity through simple automated operations.

---

### 3. SQL Injection Detection

SQL Guard is a security-focused module.

Features:

* Query scanning
* File scanning
* Pattern-based SQL injection detection
* Security report generation

The module helps identify potentially malicious SQL statements before execution.
